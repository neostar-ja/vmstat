# 📊 คู่มือ Data Store Executive Dashboard สำหรับผู้บริหาร

## 📝 ภาพรวม

Data Store Executive Dashboard เป็นแดชบอร์ดสำหรับผู้บริหารที่ออกแบบมาโดยเฉพาะเพื่อการนำเสนอข้อมูล Data Storage บนหน้าจอขนาดใหญ่ เช่น ทีวี จอมอนิเตอร์ หรือ Display ต่างๆ พร้อมฟีเจอร์ที่ทรงพลังสำหรับการนำเสนอและรายงาน

## 🎯 จุดเด่นหลัก

### ✨ การออกแบบ Modern & Professional
- **UI/UX ระดับ Enterprise**: ใช้ Material-UI (MUI) + Tailwind CSS
- **สีสันสดใส**: ใช้ Gradient และ Color Palette แบบมืออาชีพ
- **Animations**: มี Transitions และ Hover Effects ที่ลื่นไหล
- **Responsive**: รองรับทุกขนาดหน้าจอ จากมือถือถึง 4K Display

### 📺 เหมาะสำหรับการแสดงผล
- **Full Screen Mode**: สามารถขยายเต็มหน้าจอได้ด้วยการคลิกเพียงครั้งเดียว
- **One-Page View**: แสดงข้อมูลทั้งหมดในหน้าเดียว ไม่ต้องเลื่อน
- **High Contrast**: สีสันที่ชัดเจน อ่านง่าย แม้จากระยะไกล
- **Auto-Refresh**: รีเฟรชข้อมูลอัตโนมัติทุก 60 วินาที

### 📤 การส่งออกข้อมูล
- **Export เป็น PNG**: ส่งออกเป็นภาพความละเอียดสูง (Scale 3x)
- **Crystal Clear**: เหมาะสำหรับรายงาน, การนำเสนอ, และเอกสาร
- **รองรับภาษาไทย**: Font และ Text แสดงผลภาษาไทยได้สมบูรณ์

## 🚀 วิธีการเข้าถึง

### ขั้นตอนที่ 1: เข้าสู่ระบบ
1. เข้าสู่ระบบที่ `https://10.251.150.222:3345/vmstat/`
2. ใช้ Username และ Password ของคุณ

### ขั้นตอนที่ 2: ไปยังหน้า DataStores
1. คลิกที่เมนู **"DataStores"** ในแถบด้านข้าง
2. คุณจะเห็นหน้า DataStore Management

### ขั้นตอนที่ 3: เปิดแท็บผู้บริหาร
1. คลิกที่แท็บ **"ภาพรวมผู้บริหาร"** (Executive Dashboard)
2. แดชบอร์ดจะแสดงข้อมูล Data Store ทั้งหมดที่ถูกเลือกไว้

## 🎨 ส่วนประกอบของ Dashboard

### 1. Summary Cards (การ์ดสรุป)
แสดง 4 ตัวชี้วัดหลัก:

#### 📦 จำนวน Data Store
- จำนวน Data Store ทั้งหมดในระบบ
- สี: น้ำเงิน (Primary)

#### ☁️ ความจุรวม
- พื้นที่เก็บข้อมูลรวมทั้งหมด (Total Capacity)
- สี: ฟ้า (Info)

#### 📊 พื้นที่ใช้แล้ว
- พื้นที่ที่ถูกใช้งานไปแล้ว
- แสดง % การใช้งาน
- สี: ส้ม (Warning)

#### 💚 พื้นที่ว่าง
- พื้นที่ที่ยังว่างอยู่สามารถใช้งานได้
- สี: เขียว (Success)

### 2. Charts Section (ส่วนกราฟ)

#### 🥧 Pie Chart - การกระจายการใช้งาน Storage
- แสดงสัดส่วนการใช้งานของแต่ละ Data Store
- ใช้สีแยกแยะได้ชัดเจน
- มี Label แสดง % โดยตรง

#### 📊 Bar Chart - เปรียบเทียบการใช้งาน
- แสดงเปรียบเทียบ "ใช้แล้ว" vs "ว่าง" ในหน่วย GB
- แท่งแนวนอน อ่านง่าย
- Hover เพื่อดูข้อมูลละเอียด

### 3. Datastore Cards (การ์ดรายละเอียด)

แต่ละ Data Store จะแสดงเป็นการ์ดที่ประกอบด้วย:

#### Header Section
- **Status Badge**: แสดงสถานะ "ปกติ" (เขียว) หรือ "เตือน" (ส้ม/แดง)
- **Type Badge**: ประเภทของ Storage (VMFS, NFS, VS)
- **AZ Badge**: Availability Zone (DC/DR)
- **ชื่อ Data Store**: แสดงชื่อแบบเด่นชัด

#### Usage Bar (แถบการใช้งาน)
แสดงเปอร์เซ็นต์การใช้งานด้วยสี:
- 🟢 **เขียว** (0-70%): ปกติ
- 🟡 **เหลือง** (70-80%): ใกล้เต็ม
- 🟠 **ส้ม** (80-90%): ค่อนข้างเต็ม
- 🔴 **แดง** (90-100%): เกือบเต็ม/ต้องระวัง

#### Stats Grid (ตารางสถิติ)
แสดง 3 ค่าหลัก:
1. **ความจุ** (Total Capacity)
2. **ใช้ไป** (Used) - แสดงด้วยสีตามระดับการใช้งาน
3. **คงเหลือ** (Free) - แสดงด้วยสีเขียว

#### Trend Section (ส่วนแนวโน้ม)
แสดงการเปลี่ยนแปลงเมื่อเทียบกับ:
- **1 วัน**: การเปลี่ยนแปลงจากเมื่อวาน
- **7 วัน**: การเปลี่ยนแปลงจากสัปดาห์ที่แล้ว

มีไอคอนและสี:
- 📈 **แดง**: เพิ่มขึ้น (ใช้งานมากขึ้น)
- 📉 **เขียว**: ลดลง (พื้นที่ว่างเพิ่มขึ้น)

## 🖥️ ฟีเจอร์พิเศษ

### Full Screen Mode (โหมดเต็มหน้าจอ)

#### วิธีเปิด:
1. คลิกปุ่ม **Full Screen** (🔲) ที่มุมบนขวา
2. หน้าจอจะขยายเต็มจอทันที
3. เหมาะสำหรับ:
   - แสดงบนทีวี
   - การประชุม
   - NOC/SOC Display
   - Command Center

#### วิธีออก:
1. คลิกปุ่ม **Exit Full Screen** (🗗) หรือ
2. กดปุ่ม **ESC** บนคีย์บอร์ด

### Export เป็นภาพ PNG

#### วิธีใช้:
1. คลิกปุ่ม **Download** (⬇️) ที่มุมบนขวา
2. รอระบบประมวลผล (ประมาณ 2-5 วินาที)
3. ไฟล์จะถูกดาวน์โหลดอัตโนมัติ

#### คุณสมบัติของภาพ:
- **ชื่อไฟล์**: `datastore-executive-dashboard-YYYY-MM-DD.png`
- **ความละเอียด**: สูงมาก (Scale 3x)
- **คุณภาพ**: 100% (ไม่มี Compression)
- **เหมาะสำหรับ**:
  - พิมพ์เป็นเอกสาร
  - นำเสนอใน PowerPoint
  - แนบในรายงาน
  - ส่งทาง Email

### Auto-Refresh (รีเฟรชอัตโนมัติ)
- ข้อมูลจะอัปเดตอัตโนมัติทุก **60 วินาที**
- ไม่ต้องกดรีเฟรชเอง
- เหมาะสำหรับการแสดงบนจอตลอดเวลา

### Manual Refresh (รีเฟรชด้วยตัวเอง)
- คลิกปุ่ม **Refresh** (🔄) ที่มุมบนขวา
- ข้อมูลจะอัปเดตทันที

## ⚙️ การตั้งค่า Data Store ที่จะแสดง

### ขั้นตอนการเลือก Data Store:
1. ไปที่เมนู **Settings** → **Admin Settings**
2. คลิกแท็บ **"Data Store Dashboard"**
3. เลือก Data Store ที่ต้องการแสดงโดยติ๊กถูก ☑️
4. คลิกปุ่ม **"บันทึกการตั้งค่า"**
5. กลับมาที่หน้า DataStores และเปิดแท็บ "ภาพรวมผู้บริหาร"

**หมายเหตุ**: เฉพาะ Data Store ที่เลือกไว้เท่านั้นที่จะแสดงใน Executive Dashboard

## ❓ FAQ (คำถามที่พบบ่อย)

### Q: ทำไม Dashboard ว่างเปล่า?
**A**: ต้องตั้งค่าเลือก Data Store ในหน้า Admin Settings ก่อน

### Q: ข้อมูลอัปเดตบ่อยแค่ไหน?
**A**: อัตโนมัติทุก 60 วินาที หรือคลิก Refresh ได้ตลอดเวลา

### Q: Export ภาพออกมาไม่สวย?
**A**: ตรวจสอบว่าเปิด Dark Mode หรือ Light Mode แล้วพื้นหลังจะปรับตาม

### Q: Full Screen ไม่ทำงาน?
**A**: บางเบราว์เซอร์บล็อก Full Screen โดยอัตโนมัติ ให้อนุญาตในการตั้งค่า

### Q: ภาพ Export ไม่มีภาษาไทย?
**A**: ระบบรองรับภาษาไทยโดยอัตโนมัติ ตรวจสอบ Font ในเบราว์เซอร์

### Q: สามารถแชร์ Dashboard ได้หรือไม่?
**A**: ได้ แค่ส่ง URL พร้อม Login Credentials หรือ Export เป็นภาพ

### Q: ทำไมบางข้อมูลไม่แสดง?
**A**: ตรวจสอบว่า Backend Sync Service ทำงานปกติหรือไม่
